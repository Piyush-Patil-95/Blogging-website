# Blog Website

#Description
This Blog Website is a full-stack application where users can create, read, update, and delete blogs.
It includes a complete authentication system using the Passport.js library, allowing users to register, log in, and manage their personal blog posts securely.
The project demonstrates core web-development concepts such as REST APIs, sessions, validation, database operations, and protected routes.

#Features
*User registration and login using Passport.js
*Create, Read, Update, Delete (CRUD) operations for blog posts
*Secure session-based authentication
*Password hashing for user security
*Fully protected routes (only logged-in users can create or edit posts)
*Clean folder structure for backend and frontend
*Error handling and form validation
*Responsive UI for viewing and managing blogs
