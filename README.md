# Blog Website

#Description <br>
This Blog Website is a full-stack application where users can create, read, update, and delete blogs.
It includes a complete authentication system using the Passport.js library, allowing users to register, log in, and manage their personal blog posts securely.
The project demonstrates core web-development concepts such as REST APIs, sessions, validation, database operations, and protected routes.

#Features
<ul>
<li>User registration and login using Passport.js <br></li>
Create, Read, Update, Delete (CRUD) operations for blog posts <br>
Secure session-based authentication <br>
Password hashing for user security <br>
Fully protected routes (only logged-in users can create or edit posts) <br>
Clean folder structure for backend and frontend <br>
Error handling and form validation <br>
Responsive UI for viewing and managing blogs <br>
</ul>
